# NexusHub - Smart Operations Management Portal

NexusHub is a web-based operations management system designed to help organizations manage employees, work requests, resources, notifications, and user profiles efficiently.

## Features

- 🔐 User authentication and role-based access (Admin & Employee)
- 👥 Team member management
- 📝 Work request creation, assignment, and status tracking
- 📦 Resource management
- 🔔 Real-time notification system
- 🔍 Global search functionality
- 👤 Profile management with image upload
- 📊 Dashboard statistics and activity tracking

## Technologies Used

### Frontend
- React.js
- Tailwind CSS
- React Router
- Axios

### Backend
- Node.js
- Express.js
- MySQL
- JWT Authentication
- Multer (Image Upload)
